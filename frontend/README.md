# Geofencing Alert System - Frontend

## Overview

The frontend is built using Next.js, React, Tailwind CSS, Axios, and Leaflet Maps. It provides a responsive dashboard for monitoring vehicles, geofences, violations, live alerts, and map visualization.

## Features

* Dashboard Overview
* Vehicle Management View
* Geofence Monitoring
* Violation History
* Real-Time WebSocket Alerts
* Interactive Leaflet Map
* Live Vehicle Tracking
* Responsive UI

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Axios
* React Leaflet

## Pages

### Dashboard

/

Displays:

* Total Vehicles
* Total Geofences
* Total Violations

### Vehicles

/vehicles

Displays all registered vehicles.

### Geofences

/geofences

Displays active geofence zones.

### Violations

/violations

Displays geofence entry and exit events.

### Live Alerts

/live-alerts

Displays real-time WebSocket alerts.

### Map

/map

Displays:

* Geofence Polygons
* Vehicle Marker
* Live Tracking Information

## Installation

Install Dependencies

npm install

Run Development Server

npm run dev

Frontend runs on:

http://localhost:3000

## Backend Configuration

Backend API URL:

http://localhost:8080

Make sure the backend service is running before starting the frontend.
